List和Tuple类型

list是一种有序的集合

由于python是一种动态语言，因此List里面的元素可以是字符串也可以是整型
L[i]可以访问list中的第i个元素，L[-i]表示逆序查看

Tuple是另一种有序的列表，一旦创建完毕，无法进行修改

Tuple与List类似，可以正常访问，但是无法使用方法进行修改
Tuple中创建单元素，不能简单的使用t=(1)，因为python将其默认看为整数1，
需要定义t=(1,)在元素后加个逗号方能避免歧义

‘可变的Tuple’

当Tuple如图定义时，可以通过修改list来修改Tuple
Tuple中的三个元素分别为‘a’，‘b’和一个list
list里面的内容改变了，但是仍然是最初的那个list，故Tuple元素不变依然成立，若想保证Tuple中的元素永恒不变，必须确保其中的元素本身也不能变

dick与set类型

dick定义一种查找表

名字称为key，成绩称为value
list通过索引来寻找元素
dick通过key来查找value

lend()函数可以计算任意集合的大小
lend(d) = 3 一个key-value算一个

dick与list进行对比
dick查找速度快，占用内存大，dick中存放元素无序
list查找速度慢，占用内存小
作为key的元素必须不可变，由于list是可变的，因此不能作为key
而Tuple则可以作为key

更新dick

用for循环可以遍历dick

set内部的元素是无序且不重复的
创建set的形式————调用set函数，传入一个list

若传入重复的元素，则会自动去掉

访问set中的元素实际就是判断元素是否在set中，返回值为布尔类型

set与dick类似，但是不存储value，故查找速度快很多，且set中元素与dick中的key一样都是不变的
遍历set与list类似

迭代

在Python中，如果给定一个list或tuple，我们可以通过for循环来遍历这个list或tuple，这种遍历我们成为迭代（Iteration）。
· 有序集合：list，tuple，str和unicode；
· 无序集合：set
· 无序集合并且具有 key-value 对：dict

Python中，迭代永远是取出元素本身，而非元素的索引
使用 enumerate() 函数实际上将['Adam', 'Lisa', 'Bart', 'Paul']
变成了[(0, 'Adam'), (1, 'Lisa'), (2, 'Bart'), (3, 'Paul')]
迭代的每一个元素实际上是一个tuple
zip函数可以把两个list变成一个list

d. itervalues()与 d.values()都可以取出dict的value,二者的不同在于
values()把dict转换成为一个包含value的list
itervalues()依次从dict中取出value，节省了内存

items() 方法把dict对象转换成了包含tuple的list,iteritems()方法则同上
